4 Hypotheses
H1
Tool system co effect size lon nhat (Cohen's d > 0.5) trong 3 thanh phan harness doi voi resolve rate
H2
Context management co medium effect size (Cohen's d = 0.3–0.5)
H3
Harness configuration giai thich it nhat 10% resolve rate variance (η² ≥ 0.10 trong ANOVA), doc lap voi LLM backend
H4
