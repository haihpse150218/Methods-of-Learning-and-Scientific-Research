4. Gia thuyet nghien cuu

H1
Tool system co effect size lon nhat (Cohen's d > 0.5) trong 3 thanh phan harness doi voi resolve rate.
H2
Context management co effect size trung binh (Cohen's d 0.3 – 0.5).
H3
Harness configuration giai thich it nhat 20% variance cua resolve rate (eta² ≥ 0.20), doc lap voi LLM backend.
H4
Variance giua cac LLM backends giam khi harness chat luong cao hon (interaction effect co y nghia thong ke).
Trung tinh: Khong du doan con so ("30–40%"). Chi neu huong co the kiem chung → tranh confirmation bias. Du H duoc ho tro hay bac bo deu la dong gop.

9. ANOVA 2 chieu — Phan tach Harness vs LLM

"Doc lap voi LLM" khong phai nghia la harness va LLM khong tuong tac. Nghia la co the phan tach phan variance do harness vs do LLM.

Mo hinh thong ke:

Total Variance = Var(Harness) + Var(LLM) + Var(Harness x LLM) + Error

Factor A: Harness config   (9 levels: 3 tool x 3 context)
Factor B: LLM backend      (3 levels: Claude, GPT, DeepSeek)
DV:        Resolve rate

Neu Var(Harness) co y nghia    --> harness CO anh huong doc lap
Neu Var(H x LLM) co y nghia    --> co INTERACTION
eta-squared                    --> % variance cua moi factor
Dong gop ly thuyet chinh: Formalize cach phan tach harness quality khoi LLM quality. Post-hoc: Tukey HSD pairwise, Cohen's d effect sizes, Bonferroni correction cho 7 metrics.
